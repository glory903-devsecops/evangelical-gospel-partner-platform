# 01. 프로젝트 개요

## 프로젝트명
전도 파트너 플랫폼

## 서비스 성격
지역교회와 연계하여 전도 모임 신청자 모집, 지역별 공지 운영, 모임 신청 관리, 간단한 커뮤니티 운영을 지원하는 비영리 웹/앱 플랫폼

## 운영 대상 테넌트
- 안국역 전도 파트너
- 삼성역 전도 파트너
- 판교역 전도 파트너
- 다산역 전도 파트너

## 핵심 방향
- 하나의 엔진(공통 백엔드) + 4개 테넌트 구조
- Flutter로 모바일 앱과 웹앱 동시 개발
- Firebase 기반 서버리스/관리형 구조
- 1인 운영 가능성 극대화
- 최소 기능 MVP부터 시작

## 해결하려는 문제
기존 전도 모임 운영은 카카오톡, 밴드, 구글폼, 엑셀 등으로 분산되어 있어 운영 효율과 데이터 일관성이 떨어짐. 지역별 모임 운영, 신청자 관리, 공지 배포, 기본 커뮤니티 기능을 하나의 플랫폼으로 통합하려는 목적.

## MVP 핵심 기능
1. 회원가입 / 로그인
2. 테넌트(지역) 선택 또는 지정
3. 지역별 공지 게시판
4. 모임 목록 / 신청
5. 관리자 공지 등록
6. 동시 접속 제한 게이트 화면
7. 관리자 최소 페이지

## 제외 기능 (초기)
- 고급 채팅
- 결제
- 복잡한 후원 정산
- 고급 통계 대시보드
- 다국어
- 오프라인 모드
