# 10. 결과 검수 체크리스트

## 공통
- [ ] Flutter 프로젝트가 실행된다.
- [ ] Web 빌드가 된다.
- [ ] Android/iOS 빌드 준비 상태가 정리되어 있다.
- [ ] 폴더 구조가 과도하게 복잡하지 않다.

## 인증
- [ ] 회원가입 가능
- [ ] 로그인 가능
- [ ] 로그아웃 가능

## 멀티테넌트
- [ ] 사용자별 tenantId가 저장된다.
- [ ] 서로 다른 테넌트 데이터가 섞이지 않는다.

## 공지
- [ ] 공지 목록 조회 가능
- [ ] 공지 상세 조회 가능
- [ ] 운영자 이상만 공지 등록 가능

## 모임
- [ ] 모임 목록 조회 가능
- [ ] 모임 신청 가능
- [ ] 신청 취소 가능
- [ ] 내 신청 목록 확인 가능

## 접속 제한
- [ ] 허용 인원 초과 시 게이트 화면 표시
- [ ] 재확인 버튼 동작
- [ ] 입장 가능 시 자동 이동
- [ ] 앱 강제 종료 없음

## 관리자
- [ ] 운영자 지정 가능
- [ ] 테넌트별 maxConcurrentUsers 설정 가능
