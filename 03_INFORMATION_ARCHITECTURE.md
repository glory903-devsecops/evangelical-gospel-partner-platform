# 03. 정보구조 및 화면 구성

## 공통 메뉴 구조
### 비로그인
- 온보딩 / 소개
- 로그인
- 회원가입
- 비밀번호 재설정

### 로그인 후 일반 사용자
- 홈
- 공지
- 모임
- 내 신청
- 내 정보

### 운영자
- 홈
- 공지
- 모임
- 신청자 관리
- 운영 설정(최소)
- 내 정보

### 전체 관리자
- 대시보드
- 테넌트 관리
- 사용자 관리
- 운영자 권한 관리
- 접속 제한 설정
- 공지/모임 관리

## 홈 화면
- 테넌트명
- 상단 공지 요약
- 다가오는 모임 3개
- 빠른 버튼: 공지 보기 / 모임 보기 / 내 신청 내역

## 공지 화면
- 공지 목록
- 공지 상세

## 모임 화면
- 진행 예정 모임 목록
- 모임 상세
- 신청 버튼
- 신청 완료 상태 표시

## 내 신청 화면
- 신청한 모임 목록
- 상태(신청 완료 / 취소 / 마감)

## 게이트 화면
- 현재 허용 인원 표시
- 현재 접속 상태 안내
- 재확인 버튼
- 자동 재조회 문구
